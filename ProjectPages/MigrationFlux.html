def minmax(vals, min_a = 0, max_a = 1):
    Z = max(vals) - min(vals)
    out_vals = (vals - min(vals))/Z*(max_a - min_a) + min_a
    return out_vals

Introduction¶

I use the SCB API to get data over migration patterns between Swedish counties.
The data contains "OD-pairs" for migrationns aggregated by years.

Get county map¶

Since we want to show the spatial migration pattern between Swedish counties we need to create a map with each county.
The data for each county will be mapped to the centroid.

To create a map of the migration patterns we load the county data from a postgis database.
In this step we also extract the centroids of all the county polygons

The map is loaded from a PostGIS database with official Swedish county geometries.
Although we could get the centroids from PostGIS with the function ST_centroid().
Geopandas also supports returning centroid of the dataframe geometry.

Get SCB migration data¶

We use the post method where the query parameters are in a separate body than the query string. According to the documentation, an API URL should have the format:

API-NAME/API-VERSION/LANGUAGE/DATABASEID/<LEVELS>/TABLE-ID

where,

  • API-NAME: Defines the root name of the API.

  • API-VERSION: Self explanatory. It's the API version.

  • LANGUAGE: Defines the langage of the data collection. Commonly "doris"
  • DATABASE-ID: Specifies the ID of the database containing the statistical tables.
  • LEVELS: defines zero or more levels in the database.
  • TABLE-ID: Defines the ID of a statistics table.

For example:

url = "http://api.scb.se/OV0104/v1/doris/sv/ssd/START/BE/BE0101/BE0101J/InOmflytt"

Would mean that:

  • API-Name = "OV0104"
  • API-VERSION = "v1"
  • LANGAUGE = "doris

Below the migration data for 2020 is collected.

Migration size from Stockholm.¶

With this data, we can create a map showing the size of the migration from Stockholm to each respective county.
The size of the circle marker, as well as the opacity, show the size of the flow.

Getting the geometry correct took a bit of work to figure out.
The main difficulty was in getting the linestring between each county to clip at the circle boundary.
Without this, the line went to the center of the circle polygon and was visible.

Net Migration flux¶

We can also show the net migration between two OD pairs.
In this case, we look at the difference between people moving to Stockholm from each respective county and people moving from Stockholm to each respective county.
